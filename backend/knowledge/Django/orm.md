# Django ORM

The Django ORM allows developers to interact with the database using Python objects instead of SQL.

## Common Operations

Create

Model.objects.create()

Retrieve

Model.objects.get()

Filter

Model.objects.filter()

Update

object.save()

Delete

object.delete()

## Common Errors

IntegrityError

DoesNotExist

MultipleObjectsReturned

OperationalError
