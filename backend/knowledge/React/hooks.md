# React Hooks

## Definition

Hooks allow React functional components to manage state and lifecycle.

## Common Hooks

useState

useEffect

useContext

useRef

useMemo

## Common Errors

Invalid hook call

Too many re-renders

Missing dependency

State update loop

## Solutions

Call hooks only at the top level

Never call hooks inside loops or conditions

Follow dependency array rules
