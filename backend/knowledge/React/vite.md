# Vite

## Definition

Vite is a modern frontend build tool for React applications.

## Create Project

npm create vite@latest

## Install Dependencies

npm install

## Run Development Server

npm run dev

## Build Project

npm run build

## Common Errors

Failed to resolve module

Port already in use

Node version mismatch

## Solutions

Delete node_modules

Run npm install

Restart Vite server

Update Node.js

Official Documentation:
https://vitejs.dev/
